# Personal Expense Tracker

A simple and lightweight console-based application to track daily expenses and manage personal budgets.

---

## Overview

Personal Expense Tracker helps users record, organize, and review their spending habits. The application is designed to be easy to use and runs entirely in the console. It allows users to store expenses, categorize them, and monitor their financial activity in a structured way. This project was created to practice core programming concepts and basic database integration.

---

## Features

- Add new expenses with amount, date, and category  
- View all saved expenses in a list format  
- Update existing expense records  
- Delete unwanted entries  
- Category-wise expense tracking  
- Monthly budget setting and limit warnings  
- Export expense data to external files  

---

## Technologies Used

- Java (Core Java)  
- SQLite for local data storage  
- JDBC for database connectivity  

---

## Installation & Setup

1. Install Java JDK (8 or higher) on your system.  
2. Clone the repository or download the source code.  
3. Open the project in any Java-supported IDE.  
4. Make sure the SQLite JDBC driver is available in your project.  
5. Run the `Main.java` file to start the application.  

---

## How to Use

After running the program, a menu will appear in the console. Use the number options to:

- Add new expenses  
- View or search expenses  
- Update existing records  
- Delete records  
- Manage budgets  
- Exit the application  

---

## Testing

You can test the project by:

- Adding multiple expenses with valid and invalid inputs  
- Updating and deleting existing records  
- Checking budget alerts when limits are exceeded  
- Verifying database data persistence after restarting the program
